# Suggested Commands

## Development
- `uv sync` - Install dependencies
- `uv run transcribe.py <audio_file>` - Transcribe audio
- `uv run split.py <output_dir>` - Export split audio files
- `uv run edit.py <output_dir>` - Launch GUI editor
- `uv run export_edit.py <output_dir>` - Generate edit_segments.json

## System (Windows)
- `git` - Version control
- `gh` - GitHub CLI
